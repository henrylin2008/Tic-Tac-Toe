# Tic-Tac-Toe
* 2 Player game
* Player gets to select his marker (X or O), second player gets the other marker
* System will auto select which player go first
* Winning the game:
  * entire row
    * row 1: 1-2-3
    * row 2: 4-5-6
    * row 3: 7-8-9
  * entire column
    * column 1: 1-4-7
    * column 2: 2-5-8
    * column 3: 3-6-9
  * cross row and column
    * position: 1-5-9 or 3-5-7
